CloakFrame 1.10.3 replaces the built-in smiley sticker with custom image
overlays for detected faces and license plates.

### New features
- Select a PNG, JPEG, BMP, TIFF, or WebP image and place it over every
  detected region
- Use the same custom image in photos, videos, style previews, and image
  review previews
- Preserve custom-image transparency, including natural blending of
  partially transparent pixels with the original image

### Improvements
- Saves the selected custom-image path between launches
- Loads a bounded image snapshot before processing so later file changes do
  not affect an active photo or video run
- Adds custom-image coverage for 8-bit and 16-bit photos and the parallel
  video masking pipeline
- Keeps the custom image's aspect ratio and aligns face overlays to SCRFD
  eye and mouth landmarks
- Interpolates and smooths overlay position, size, and tilt along video face
  tracks for steadier motion
- Uses premultiplied-alpha transforms to avoid dark fringes around transparent
  custom images

Transparent pixels leave the original image visible. Use an opaque image when
the detected region must be fully covered.
